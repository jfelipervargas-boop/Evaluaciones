# Programa: Mayor Magnitud de Puntos

Este programa en C++ permite:

- Ingresar N puntos en el plano.
- Calcular la magnitud (distancia al origen) de cada punto.
- Determinar cuál tiene la mayor magnitud.
- Mostrar el índice y las coordenadas del punto correspondiente.

## Cómo compilar

g++ main.cpp -o programa

## Cómo ejecutar

./programa
